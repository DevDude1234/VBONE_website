The Birth of H8RZ
H8RZ was the first crypto token autonomously launched by an AI and occurred on December 10, 2024 (via an AI Agent, Root, developed by Cyphomancer).
The Cyphomancer dev asked Root in a public Telegram chat to launch a coin with a theme about people that do not own the Cyphomancer token $42. Root then created a token name, ticker, description, token logo image and launched the token on pump.fun via an API call (pumpportal) all autonomously.

The Details of H8RZ Launch on December 10, 2024
The Cyphomancer Dev tags Root in a public Telegram post requesting the launch of a token
Root creates token name, ticker and description. Root assumed that the token socials (X, Telegram, website) were for cyphomancer since the dev did not provide them and the theme centered upon Cyphomancer.
Root sends API call to ChatGPT 4o to generate a token image based upon the token info.
Root sends API call to Pumpportal to create token using Roots Solana wallet (failed 2 times and successful upon the 3rd auto-retry)
Root receives confirmation of successful transaction

RE-Discovery of H8RZ
During the launch of H8RZ, the historical nature of this feat was not realized. Initially, this event was celebrated as just a cool feature of the only uncensored AI Agent in Crypto (Root).
The H8RZ token was quickly forgotten in the wake of a bullrun where attention was diluted among 1000s of coins. It was not until the launch of $DRB by the X AI (Grok), about 100 days later, that attention shifted toward AI-launched tokens.
Members of the Cyphomancer community then realized the significance of what H8RZ actually was the first token launched by an AI. It is important to note that the token GOAT associated with Truth Terminal was not launched by AI. GOAT was launched by the Truth Terminal dev (human). Truth Terminal simply provided a suggested name for a hypothetical crypto token.
Therefore, H8RZ was the first token in the world autonomously launched by an AI.
